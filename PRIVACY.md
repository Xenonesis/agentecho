Privacy Policy — Pinmark

Pinmark does not collect, sell, or share personal information.

Data processing
Pinmark runs locally in the user’s browser. When activated by the user, it may read webpage element information (such as tag name, text content, classes, IDs, and page URL) only to generate the user’s feedback report.

Storage
User annotations and feedback notes are stored locally using the Chrome storage API so they can persist on the same website/page. This data remains on the user’s device unless the user manually copies and shares it.

No remote servers
Pinmark does not send data to external servers and does not use analytics, tracking, or remote code.
